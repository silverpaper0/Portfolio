# 나만의 포트폴리오 템플릿 만들기

- Start date : 2021. 09. 30

## 프로젝트 개요

- 디자인 전공 park 와 컴퓨터 전공 choi 가 협업하여  미니 프로젝트를 진행하고자 합니다.

   html,css, js를 사용해서 누구나 사용할 수 있는 포트폴리오 템플릿 하나를 만들어 볼려고합니다.

   2021 올해 안으로 완성도 높은 포트폴리오 템플릿 사이트를 만드는 것을 목표 입니다.
